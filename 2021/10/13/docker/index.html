docker

docker

  1. 运行容器
    1. docker run
      1. -t 指定一个终端
      2. -i 使用输入
      3. -p 运行映射的端口
      4. -P 随机映射到主机的端口
      5. -d 后台运行
        1. 想要后台运行后进入
          1. docker attach
          2. docker exec
            1. docker -it xxx /bin/bash
      6. —name 容器的名称
      7. -v 挂在数据卷
  2. 容器导出
    1. docker export
      1. docker export xxx >  xx.tar
  3. 查看容器的状态
    1. 映射的端口
      1. docker port
    2. 查看日志
      1. docker logs
        1. -f 标准输出
    3. 查看容器信息
      1. docker ps
  4. 镜像操作
    1. 查看镜像
      1. docker images
    2. 查找镜像
      1. docker search
    3. 获取镜像
      1. docker pull
    4. 删除镜像
      1. docker rmi
    5. 更新镜像
      1. 创建容器 run
      2. apt-get update
      3. docker commit
    6. 构建镜像
      1. docker build
        1. -t 目标镜像名
        2. . dockerfile位置
        3. docker build -t xxx ./
      2. 添加标签
        1. docker tag
    7. 推送镜像
      1. docker tag xxx
      2. docker push
  5. 网络
    1. docker network create
      1.  bridge 隔离各个容器，让每个容器的端口都是隔离的
      2. overlay 虚拟网络
  6. dockerfile
    1. from 从xxx地方复制需要的组建
    2. run 拼接shell命令 docker build的时候执行
    3. copy 存储位置，目标位置
    4. add 同copy，会解压
    5. cmd 在docker run的时候执行
    6. env 环境变量
    7. arg 只在build的时候生效
    8. volume 挂在数据卷
    9. label 标签
